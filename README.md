# Hospital Management System

A full-stack Hospital Management System built using Spring Boot, HTML, CSS, JavaScript, and MySQL/H2 Database.

---

## Features

- Patient Registration
- Doctor Dashboard
- Reception Dashboard
- Appointment Management
- Billing System
- Prescription Generation
- Queue Management
- Medical Reports
- Authentication/Login System

---

## Tech Stack

### Backend
- Java
- Spring Boot
- Spring MVC
- Spring Data JPA

### Frontend
- HTML
- CSS
- JavaScript

### Database
- MySQL / H2

### Tools
- Git
- GitHub
- Maven
- Docker
